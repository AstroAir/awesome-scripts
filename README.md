一个实用的油猴脚本合集，包含 GitHub 增强和 Linux.do 论坛工具。

## 🔧 安装方法

### 前置要求

请先安装以下任一浏览器扩展：

- [Tampermonkey](https://www.tampermonkey.net/) (推荐)
- [Violentmonkey](https://violentmonkey.github.io/)
- [Greasemonkey](https://www.greasespot.net/) (仅 Firefox)

### 安装步骤

1. 安装上述浏览器扩展
2. 点击下方脚本链接进入脚本页面
3. 点击 "Raw" 按钮查看源码
4. 油猴扩展会自动弹出安装确认窗口
5. 点击 "安装" 即可

> 💡 **提示**: 也可以直接复制脚本内容，在油猴扩展中新建脚本并粘贴。

## 📖 使用说明

### GitHub 增强工具栏

- 访问任意 GitHub 页面即可看到增强按钮
- 支持一键跳转到 Github.dev、DeepWiki 和 ZreadAi

### GitHub Trending Button Enhanced

- 在 GitHub 导航栏显示 Trending 按钮
- 支持按编程语言筛选
- 可收藏常用语言
- 快捷键: `Alt+T` 快速打开

### GitHub Fold Files Enhanced

- 在仓库文件列表页面可折叠文件夹
- 支持全部折叠/展开
- 折叠状态会自动保存

### GitHub Fold About Sidebar

- 点击 About 区域旁的箭头即可折叠
- 折叠状态会自动保存

### Linux.do 自动标记已读

- 打开脚本面板选择阅读模式
- 支持多种阅读策略
- 可配置通知和声音提醒

### Linux.do 帖子导出器

- 在帖子页面显示导出按钮
- 支持 JSON 和 HTML 两种格式
- 可选择是否嵌入图片
