## GitHub Trending Button Enhanced

**版本**: v2.0.0 | **文件**: `github-trending-button.user.js`

在 GitHub 导航栏添加增强版 Trending 按钮，快速访问热门项目。

### 功能特性

- ✅ 在导航栏显示 Trending 按钮
- ✅ 下拉菜单快速筛选编程语言
- ✅ 收藏常用语言，快速访问
- ✅ 记录最近访问的语言
- ✅ 支持按时间范围筛选（今日/本周/本月）
- ✅ 语言搜索功能
- ✅ 键盘快捷键支持
- ✅ 自定义设置

### 快捷键

| 快捷键 | 功能 |
|-------|------|
| `Alt + T` | 直接打开 Trending 页面 |
| `Alt + Shift + T` | 打开/关闭下拉菜单 |
| `Esc` | 关闭下拉菜单 |

### 下拉菜单使用

1. **点击按钮**：直接跳转到 Trending 页面
2. **悬停按钮**：显示下拉菜单
3. **搜索语言**：在搜索框输入语言名称
4. **收藏语言**：点击语言旁的星号图标
5. **选择时间范围**：点击 Today/Week/Month

### 设置选项

通过油猴菜单或下拉菜单的设置按钮访问：

- **显示收藏区域**：是否显示收藏的语言
- **显示最近访问**：是否显示最近访问的语言
- **最近访问数量**：显示多少个最近访问的语言
- **默认时间范围**：默认的 Trending 时间范围

## GitHub Fold Files Enhanced

**版本**: v1.0.1 | **文件**: `github-fold-files.user.js`

在 GitHub 仓库文件列表中添加折叠功能，方便浏览大型项目。

### 功能特性

- ✅ 折叠/展开单个文件夹
- ✅ 一键折叠/展开所有文件夹
- ✅ 平滑动画效果
- ✅ 折叠状态自动保存
- ✅ 鼠标悬停预览折叠内容

### 使用方法

1. **折叠单个文件夹**：点击文件夹行左侧的箭头图标
2. **展开单个文件夹**：再次点击箭头图标
3. **全部折叠**：点击工具栏的"全部折叠"按钮
4. **全部展开**：点击工具栏的"全部展开"按钮

### 状态保存

脚本会自动保存每个仓库的折叠状态，下次访问时自动恢复。

## GitHub Fold About Sidebar

**版本**: v1.0.0 | **文件**: `github-fold-about.user.js`

折叠 GitHub 仓库页面右侧的 About 区域，节省屏幕空间。

### 功能特性

- ✅ 一键折叠/展开 About 区域
- ✅ 平滑动画效果
- ✅ 折叠状态自动保存
- ✅ 适配亮暗色主题

### 使用方法

点击 About 区域右上角的折叠箭头即可切换折叠状态。

## GitHub 增强工具栏

**版本**: v2.1.1 | **文件**: `github-enhanced-toolbar.user.js`

在 GitHub 顶部工具栏添加实用的快捷按钮。

### 功能特性

- ✅ **Github.dev**：在 VS Code 网页版中打开当前仓库
- ✅ **DeepWiki**：在 DeepWiki 中查看仓库文档
- ✅ **ZreadAi**：使用 AI 分析仓库代码
- ✅ 完美适配亮暗色主题
- ✅ 根据按钮数量自动切换显示模式

### 按钮说明

| 按钮 | 功能 | 目标 URL |
|-----|------|----------|
| Github.dev | 在线编辑器 | `github.dev/{owner}/{repo}` |
| DeepWiki | 文档查看 | `deepwiki.com/{owner}/{repo}` |
| ZreadAi | AI 分析 | `zread.ai/github/{owner}/{repo}` |

### 显示模式

- **图标+文本模式**：按钮较少时显示图标和文本
- **纯图标模式**：按钮较多时只显示图标（悬停显示提示）

### 适用页面

脚本在以下页面显示按钮：

- 仓库主页：`github.com/{owner}/{repo}`
- 文件浏览页：`github.com/{owner}/{repo}/tree/...`
- 文件查看页：`github.com/{owner}/{repo}/blob/...`
- Pull Request 页：`github.com/{owner}/{repo}/pull/...`
- Issues 页：`github.com/{owner}/{repo}/issues/...`

## 常见问题

### Q: 脚本在某些页面不生效？

A: GitHub 使用 SPA（单页应用）架构，页面切换时脚本可能需要重新初始化。尝试刷新页面。

### Q: 按钮样式与 GitHub 主题不匹配？

A: 脚本会自动检测 GitHub 主题并适配。如果出现问题，请尝试切换主题后刷新页面。

### Q: 如何禁用某个功能？

A: 可以通过油猴扩展的脚本管理界面禁用特定脚本。
