## Linux.do 自动标记已读 Ultimate

**版本**: v4.0 | **文件**: `linuxdo-autoreader.user.js`

自动将论坛帖子及回复标记为已读，支持多种运行模式和真实行为模拟。

### 功能特性

- ✅ 自动浏览并标记帖子为已读
- ✅ 多种运行模式可选
- ✅ 真实用户行为模拟
- ✅ 可拖动的浮动控制面板
- ✅ 实时进度显示
- ✅ 详细日志记录
- ✅ 快捷键支持
- ✅ 参数自定义

### 运行模式

| 模式 | 速度 | 安全性 | 描述 |
|-----|------|--------|------|
| ⚡ 极速模式 | 最快 | 低 | 最快速度处理，适合测试 |
| 🚀 快速模式 | 快 | 中 | 较快速度，有基本延迟 |
| ⚖️ 标准模式 | 中 | 中高 | 平衡速度和安全性（推荐） |
| 🎭 仿真模式 | 慢 | 高 | 模拟真实用户行为 |
| 👻 隐身模式 | 最慢 | 最高 | 最接近真实用户，最安全 |

### 快捷键

| 快捷键 | 功能 |
|-------|------|
| `Alt + S` | 开始/停止自动阅读 |
| `Alt + P` | 暂停/恢复自动阅读 |

### 参数配置

在设置面板中可配置：

| 参数 | 默认值 | 描述 |
|-----|--------|------|
| 并发数 | 3 | 同时处理的帖子数量 |
| 每批数量 | 10 | 每批处理的帖子数量 |
| 批次间隔 | 5000ms | 批次之间的等待时间 |
| 回复间隔 | 500ms | 标记回复之间的间隔 |
| 最大回复数 | 100 | 每个帖子最多处理的回复数 |

### 行为模拟

脚本模拟的真实用户行为：

- 🖱️ 随机滚动页面
- ⏱️ 随机阅读停留时间
- 📜 渐进式滚动
- 🎲 随机操作间隔
- 👆 模拟鼠标移动

### 注意事项

> ⚠️ **重要提示**
>
> 1. 请合理使用，避免过于频繁的操作
> 2. 建议使用标准或仿真模式
> 3. 首次使用建议先用少量帖子测试
> 4. 脚本仅供学习研究使用

## Linux.do 帖子导出器

**版本**: v2.1.0 | **文件**: `linuxdo-post-export.user.js`

将 Linux.do 论坛帖子导出为 JSON 或 HTML 格式。

### 功能特性

- ✅ 导出为 JSON 格式（结构化数据）
- ✅ 导出为 HTML 格式（美观可读）
- ✅ 可选嵌入图片为 Base64
- ✅ 支持 8 种界面语言
- ✅ 响应式 HTML 样式
- ✅ 支持打印优化

### 支持语言

| 语言 | 代码 |
|-----|------|
| 简体中文 | zh-CN |
| English | en |
| 日本語 | ja |
| 한국어 | ko |
| Español | es |
| Français | fr |
| Deutsch | de |
| Русский | ru |

### 使用方法

1. 打开要导出的帖子页面
2. 点击页面上的"导出"按钮
3. 选择导出格式（JSON/HTML）
4. 选择是否嵌入图片
5. 点击确认开始导出
6. 文件将自动下载

### 导出内容

导出的数据包括：

- **帖子信息**：标题、作者、发布时间、分类
- **帖子内容**：正文、图片、链接
- **回复列表**：所有回复的内容和元信息
- **用户信息**：用户名、头像、等级

### JSON 格式示例

```json
{
  "title": "帖子标题",
  "author": {
    "username": "用户名",
    "avatar": "头像URL",
    "level": 3
  },
  "createdAt": "2024-01-01T12:00:00Z",
  "category": "分类名",
  "content": "帖子正文...",
  "replies": [
    {
      "author": {...},
      "content": "回复内容",
      "createdAt": "...",
      "likes": 10
    }
  ],
  "metadata": {
    "exportedAt": "...",
    "version": "2.1.0"
  }
}
```

### HTML 导出样式

HTML 导出文件特性：

- 🎨 美观的现代化样式
- 📱 响应式设计，适配各种屏幕
- 🖨️ 打印优化，方便打印成 PDF
- 🌙 自动适配系统暗色模式
- 🖼️ 图片懒加载优化

### 图片嵌入

选择"嵌入图片"选项时：

- 图片将转换为 Base64 格式
- 导出文件可离线查看
- 文件体积会增大
- 适合归档保存

## 常见问题

### Q: 导出的帖子图片显示不了？

A: 如果不选择嵌入图片，导出的文件需要联网才能加载图片。建议选择"嵌入图片"选项。

### Q: 自动阅读脚本会被检测到吗？

A: 脚本已尽可能模拟真实用户行为，但仍建议使用仿真或隐身模式，并避免长时间连续运行。

### Q: 如何更改界面语言？

A: 帖子导出器会自动检测浏览器语言。也可以通过脚本设置手动选择语言。
